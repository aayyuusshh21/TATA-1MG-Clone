# To Run Locally
## Clone the project
```bash
  git clone https://github.com/aayyuusshh21/screeching-poison-8463.git
```

Go to the project directory

```bash
  cd screeching-poison-8463
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  json-server --watch db.json --port 3001
```

```bash
  npm run start
```
